Data Storage Converter

Convert between data storage units: bits, bytes, KB, MB, GB, TB, PB, and binary units (KiB, MiB, GiB, TiB).

Comparison

Decimal (SI)

1 KB = 1,000 bytes. 1 MB = 1,000,000 bytes.

Binary (IEC)

1 KiB = 1,024 bytes. 1 MiB = 1,048,576 bytes.

Real world

Hard drives use decimal (GB/TB). RAM uses binary (GiB).

How it's calculated

Decimal: ×1,000 per step. Binary (KiB/GiB): ×1,024 per step. Uses the UnitEngine storage conversion table.
